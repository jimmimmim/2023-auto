# 자율주행로봇 모니터링 시스템 MVP

## 설치 및 실행
```
npm install
npm start
```
사이트가 자동으로 열리지 않을 경우 <a href="http://localhost:3000/" target="_blank">http://localhost:3000/</a> 로 직접 접속하시어 실행 가능합니다.
